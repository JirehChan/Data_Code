# Data_Code

- 本仓库对应个人博客中data系列的代码实现。

## 1. 介绍

- 机器学习、数据挖掘、统计学习方法、自然语言处理、深度学习、数据分析、可视化...
- 这些好多的内容，相互穿插、相互交织，有时学起来让人显得无从下手，可能看完了一门精品课程，看完了一本经典教材，然后还是觉得有所不足（或许一开始看懂了，后来学了门新课又开始糊涂了）。
- 为了自己知识的扎实，所以用这个系列开始来补足知识的不足。
